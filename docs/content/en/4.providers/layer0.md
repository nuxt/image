Layer0 provides a simple HTTP service for optimizing images.

## Modifiers

Layer0 supports the following modifiers: `height`, `width` and `quality`

**Example:**

```html
<nuxt-img provider="layer0" src="remote-url" width="200" height="200" quality="80" />
```
